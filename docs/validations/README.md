## How To Use These Files
1. Implement validation in the frontend for user guidance.
2. Mirror all critical validation in the backend for enforcement.
3. Use these files during QA to confirm failure handling is correct.

## Completion Standard
Validation work is complete only when every documented rule is enforced, every documented failure is surfaced correctly, and no critical rule exists only on the client side.
