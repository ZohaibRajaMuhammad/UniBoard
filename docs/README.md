# UniBoard Presentation Docs

This folder contains the presentation source and the two spoken scripts built from it.

## Files

- `uniboard-presentation-master.md` - master document and source of truth
- `presentation-script-part-1.md` - login, dashboard, saved posts, notifications, search
- `presentation-script-part-2.md` - rooms, planner, analytics, knowledge base, AI, admin

## How to use

1. Read the master document first.
2. Use script part 1 for the first half of the live demo.
3. Use script part 2 for the rest of the system.
4. Keep the wording grounded in the exact product behavior described here.
