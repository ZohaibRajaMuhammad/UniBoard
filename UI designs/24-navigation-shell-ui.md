# Navigation Shell UI

## Covered Areas
- Desktop sidebar
- mobile floating bottom navigation
- page titles and shell-level context
- room list navigation

## Sidebar Character
- Stable and elegant, never bulky
- Primary product anchor for authenticated desktop use
- Should feel like a premium operating column rather than a basic menu

## Sidebar Composition
- brand mark at top
- global nav items
- room list section
- user context block

## Mobile Nav Character
- Floating rounded capsule
- compact, centered, premium
- only core tabs visible

## Active State Rules
- Active routes must be obvious
- Unread indicators must not overpower nav labels
- Room active state should remain distinct from global nav state

## Responsive Rules
- Sidebar hides below 768px
- Mobile nav appears only in app mode on small screens
- Shell spacing must adapt without collapsing hierarchy
